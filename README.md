# flight

## Requirements

+ python
+ [Google Maps for Python] (https://github.com/googlemaps/google-maps-services-python google-maps-services-python)

## architecture
+ Flask - server side
+ BackboneJS - client side

## instructions
+ Download
+ cd into flight directory
+ python server.py
+ go to http://127.0.0.1:8000/ to see web page
